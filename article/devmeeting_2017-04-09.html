DevMeeting 2017-04-09

Submitted by aerbax on Sun, 04/09/2017 - 16:22

Highlights[ ]

  • No major issues from the 0.10.3.1 release.
  • 0.10.3.2 will likely be release before the April 14th hardfork. This will address some GUI issues, and a bug that mostly affects exchanges.
  • Discussion regarding renaming of the monero toolset. ie. Do we want to continue with "GUI" or "core" or something else. Tabled.
  • The Monero website redesign is continuing forward. It sounds like we're nearing a final design.
  • MoneroMooo closed several bugs. Fluffy will be closing them in the next few days.
  • Technical discussion of Raising the mandatory ringsize in the Sept 2017 hardfork
    • Would possibly implement a static set of ring sizes for all users - which could increase anonymity.
    • To further illustrate, if a user continually used a non-commonly used number like "27", it would be easier to demask.
    • The static ring size numbers are all up for discussion. 10,20,50,100 was brought up.
    • An MRL paper on the topic may need to be created.
  • kenshi84's disposable addresses patch may be pulled in soon. Fluffy needs to re-review.
  • April 23rd, 2017 is next meeting.
