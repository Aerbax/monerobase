DevMeeting 2017-02-05

Submitted by aerbax on Sun, 02/05/2017 - 16:25

Highlights[ ]

  • ~Two months since the mast meeting
  • Merged commits
    • batched txns for node syncing - reduces I/O load (2x-3x improvement on spinning disks). This also helped fix some corruption seen on Windows.
    • Fluffyblocks may soon be enabled on MainNet - as TestNet is showing no issues
    • MoneroMooo has a new logging implementation
  • Due to a lack of Github permission robustness, there's a need to setup a bot to handle closing issues and notifications.
  • Some discussion regarding block sizes and fees..no conclusion.
  • GUI remains in beta
