DevMeeting 2017-11-19

Submitted by aerbax on Sun, 11/19/2017 - 16:09

Highlights[ ]

  • The MRL team has thoroughly investigated BulletProofs
    • Roughly a drop-in replacement with 10% the size of original proofs
    • Roughly the same speed.. perhaps a teensy bit faster
    • Work will begin shortly to convert the Java test code to C/C++
  • MultiSig is progressing.
  • A variety of bugfixes were merged (hyc counts 72 merged pull requests)
    • Disguising password lengths during user input
    • Database locking updates
    • BSD fixes
    • A fix for show_transfers missing an incoming input
  • Hardware
    • Team has grown to 20 people
    • Milestone #1 complete
    • A prototype will be shown at this weeks Monero Munich Meeting
  • monero-core has been renamed to monero-gui
    • Theming is being worked on: https://i.imgur.com/89YFUj7.png
    • Work is being down to make the GUI more responsive.
  • The Monero Integrations team is ready to release a Magento plugin. Work is ongoing for OpenCart
