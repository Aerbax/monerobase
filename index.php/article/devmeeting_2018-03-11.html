DevMeeting 2018-03-11

Submitted by aerbax on Sun, 03/11/2018 - 16:03

Highlights

  • Bulletproofs audit funding is complete. Three disparate audits will take place.
  • Moneromooo has implemented into the GUI some of the mitigations against the key image reuse attacks
  • Ledger code was merged into the Monero codebase. This doesn't mean Ledger is ready to go, as Ledger still has to implement things into their software.
  • Stagenet created.
  • surae will announce MAGIC soon. An educational non-profit focused on students studying cryptocurrencies and building schools.
  • MoneroMooo's Borromean Signature Speedup(~10+%) will either go into the release at the end of March...or a point release after that.
  • Some work is being done to deprecate Payment ID's. This isn't expected to go live until *at least* September - and it will not remove them altogether (for the foreseable future) for compatibility reasons.
  • Endogenic and vtnerd are working on "libMoneroConsensus"
    • A library that other programs/tools can use to get easier access to core crypto and consensus for transactions
    • Lessens the amount of things a new app would have to "import".
    • They need all the help they can get. ATTN potential developers looking for something to help with
  • Panic! No release name has been identified!
    • It might be Lithium Luna
