DevMeeting 2018-02-25

Submitted by aerbax on Sun, 02/25/2018 - 16:38

Highlights[ ]

  • Ledger wallet code is getting much closer to being merged.
    • Will need to work closely with the Ledger team to finalize.
  • March Fork (protocol upgrade)
    • Waiting for vtnerd's PoW changes. CUDA and CPU are pushed. OpenCL is still in testing.
    • There will be weekly dev meetings prior to the fork, and one after it. Sundays at 17:00 UTC.
  • Discussion on changing the default ringsize
    • https://github.com/monero-project/monero/issues/3035
    • Surae (MRL) suggests: "i vote fixed ring size of 7, or, failing that, a min ring size 7, or, failing that, a default ring size 7 with our fixed ring size 5. that's my order of preference."
    • Sarang (MRL) agrees.
    • Consensus points to Surae's second option - setting the ringsize to a *minimum* of 7.
    • SGP's research:https://docs.google.com/spreadsheets/d/1iLa_yklutjHqn_DrOlO_eTb00l4YDAezijX2J5r6P14/edit?usp=sharing
