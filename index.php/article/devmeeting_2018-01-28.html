DevMeeting 2018-01-28

Submitted by aerbax on Sun, 01/28/2018 - 16:06

Highlights[ ]

  • Bulletproofs can use "batch verification" which can greatly speed things up
    • Perhaps a reduction to just ~15% of the normal time complexity
  • If you're interested in Monero Debian packages, please join the discussion at https://github.com/monero-project/monero/issues/2395
  • Monero GUI
    • Merged in sub-addresses.
    • Removed the ability for a user to generate payment id's and integrated addresses
    • There is a new "--bootstrap-daemon-address" flag that hopes to speed up syncs
  • March 2018 Hardfork
    • As previously noted, Bulletproofs need additional study before a full public implementation. As such, they will not be included in the March 2018 HF.
    • There are plans to lower the fee model. How it will impact the network is being studied.
