DevMeeting 2017-06-25

Submitted by aerbax on Sun, 06/25/2017 - 16:18

Notes[ ]

  • Fluffypony was absent for the meeting, so it's a little more informal than usual.

Highlights[ ]

  • twinget is nearly complete on the ZeroMQ RPC Daemon/Server implementation. The Wallet implementation will be further down the road.
  • vtnerd has been working on
    • Rapid importing for MyMonero backend
    • x86 Assembly(ASM) wallet import code has been live for about a month.
    • Faster,Better, Stronger backend mempool work
  • Jaquee is working on MyMoner/OpenMonero Lightwallet support in the official GUI
    • This work extends to the iOS and Android clients as well.
  • kenshi84 is working on wallet subaddresses. This is nearly ready for additional review and testing
    • Does not require a fork. Only a wallet change.
  • Discussions are ongoing for increasing the default ring size.
  • From the Kovri meeting, rehrar has completed the Monero website redesign and is waiting for it to be deployed.
