DevMeeting 2017-03-26

Submitted by aerbax on Sun, 03/26/2017 - 16:23

Highlights[ ]

  • 0.10.3 released, but a new patch build is coming(0.10.3.1) to fix a cumulative block size bug.
  • 0.10.3 has Smart Mining for Windows and Linux. Mac is not operational yet.
  • FireIce-UK's blockchain speedup proposal Two parts:
    • Part#1: Crypto Optimization
    • Part#2: Parallelism Optimization
      • Concensus in the meeting is to prioritize Part#2 first as it is the biggest "bang for the buck".
  • tewinget is progressing with ZeroMQ. Working on merging today.
  • Discussion of how to speed up a wallet's sync by potentially asking other daemon's for information. #605
    • This would require trust of the remote node, but as the local node syncs, that trust would be verified.
    • There is going to be A LOT of discussion around this.
    • Kovri may be an ideal solution.
  • The project needs more unit tests, but requiring one for a pull request is not acceptable.
